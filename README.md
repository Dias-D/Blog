# BLOG

This is a small API made in Ruby on Rails, ​​with post management per user as well as the possibility of authentication

## System settings: 
* Ruby: 3.0
* Rails: 7.0.8
* Postgres: 14.2
* Docker

## How to install
* Clone this project and move to this folder:
